git add でおこなわれていること

  • <git init>すると、そのディレクトリ配下に.gitディレクトリが作成される
  • ここで作成した.gitディレクトリ＝ローカルリポジトリ
  • <git add　sample.txt>すると、ステージングエリアに移動
  • リポジトリの配下にindexという謎ファイル
    .git/objectの配下には謎ディレクトリが作成される。
  • 詳しくは、ls -a すると、隠しディレクトリをみれるよ。
  • git add すると、変更内容は.git/indexに記録されている
  • .git /objectsに新しいオブジェクトに追加されている
